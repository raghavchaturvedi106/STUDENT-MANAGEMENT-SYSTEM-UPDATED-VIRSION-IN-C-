# 🎓 Student Management System (C Language)

A console-based Student Management System developed in C that allows users to manage student records efficiently. This project demonstrates the practical implementation of core C programming concepts such as Structures, Functions, Arrays, File Handling, and CRUD (Create, Read, Update, Delete) operations.

Unlike the initial version, this updated version stores data permanently using file handling and automatically loads student records whenever the program starts.

---

## 🚀 Features

- ➕ Add New Student
- 📋 Display All Students
- 🔍 Search Student by Roll Number
- ✏️ Update Student Details
- ❌ Delete Student Record
- 💾 Save Records to File
- 📂 Automatically Load Data on Program Startup
- 🚫 Duplicate Roll Number Validation
- 📊 Display Student Information in a Clean Format
